# BookMyGlam

BookMyGlam is a web application that allows users to book beauty services at home.

## Features

- Browse a list of available beauty services.
- Book an appointment for a service.
- View the status of your booking.

## Tech Stack

- HTML
- CSS
- JavaScript
- Supabase

## Running the Project

1. Clone the repository.
2. Create a `.env` file in the root of the project and add the following environment variables:
    - `SUPABASE_URL`
    - `SUPABASE_KEY`
3. Open the `index.html` file in your browser.
